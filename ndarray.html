Class: ndarray

ndarray(buffer, shape, dtype, offsetopt, strideopt, checkopt) → {ndarray}

new ndarray(buffer, shape, dtype, offsetopt, strideopt, checkopt) → {ndarray}

Array object to represent a multidimensional, homogeneous array. This is not supposed to be called by a user. Use np.array or np.asarray to create a new array from a buffer/Array. For developers, this is a function constructor so make sure to use the new operator.
